# 🎓 Student Management System

A comprehensive Java-based Student Management System with full CRUD operations, grade calculation, and file persistence.

## ✨ Features

- **Student Management**
  - Add new students
  - View all students
  - Search students by ID/Name/Email
  - Update student information
  - Delete students

- **Course Management**
  - Add courses
  - View all courses
  - Course code and credits

- **Grade Management**
  - Assign grades to students
  - View student grades
  - Calculate GPA automatically
  - Semester tracking

- **Reports**
  - View all students
  - View all courses
  - GPA report for all students

## 🚀 Tech Stack

| Category | Technology |
|----------|-----------|
| Language | Java 17 |
| Build Tool | None (Single File) |
| Database | In-Memory (ArrayList/HashMap) |
| Version Control | Git & GitHub |
